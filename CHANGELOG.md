### Added
- Zero-configuration setup with auto-detection
- Universal prompts that work with all AI IDEs
- MCP server integration
- CLI tools for initialization and test execution
- Project type detection (React, Vue, Angular, .NET, Node.js)
- IDE auto-configuration (Cursor, VS Code, JetBrains)
- Lightweight package design (< 5MB target)
- Tree-shakeable ESM modules
- Lazy loading for optional features

### Features
- API test generation prompts
- UI test generation prompts
- E2E test generation prompts
- Smart defaults for all configurations
- Optional config file support
