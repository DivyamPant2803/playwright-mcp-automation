# Quick Start Guide

Get started with `@playwright-mcp/automation` in 3 steps!

## Step 1: Install

```bash
npm install -D @playwright-mcp/automation
```

## Step 2: Initialize

```bash
npx playwright-mcp-init
```

This auto-detects your project and sets everything up.

## Step 3: Use with AI

In your AI IDE (Cursor, VS Code + Copilot, etc.), ask:

```
Using @playwright-mcp/automation, generate API tests for /api/users
```

That's it! The AI will generate and run tests using Playwright MCP.
